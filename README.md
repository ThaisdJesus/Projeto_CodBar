# Projeto_CodBar
# Sistema CRUD para Bar  Projeto desenvolvido para praticar desenvolvimento web utilizando HTML, CSS e JavaScript. 
## Funcionalidades 
- Cadastro de pedidos
- Listagem de pedidos
- Atualização de status
- Remoção de pedidos
- Pág ADM
- Identificação de mesas
## Tecnologias utilizadas 
- HTML5
- CSS3
- JavaScript
